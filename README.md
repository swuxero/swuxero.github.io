# deepfakeXERO

> **바종설 4팀 XERO** — 딥페이크 성범죄 피해자를 위한 웹 정보 플랫폼 + 체험형 미니게임

팀 XERO는 **성범죄(Sex crime)를 제로(Zero)로** 만들겠다는 단호한 의지를 담아 탄생했습니다.  
딥페이크 성범죄라는 어두운 현실 속에서 피해자들이 막막하고 두렵게 느낄 수 있는 복잡한 대응 절차를,  
**탭 내 미니게임 시뮬레이션**을 통해 쉽고 부담 없이 익힐 수 있도록 돕습니다.

단순한 정보 전달을 넘어, 피해자가 직접 시뮬레이션을 통해 체계적인 대응 방법을 익히고  
스스로의 권리를 당당히 찾을 수 있도록 지원하는 것 — 그것이 XERO가 추구하는 진정한 **'Zero'의 가치**입니다.
